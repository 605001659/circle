Mon, Sep 5, 2022 Hello Hugo aaa bbb ccc cd D:\hugo 生成站点 ./hugo.exe new site D:\hugo 创建文章 创建一个 about 页面 hugo new about.md about.md 自动生成到了 content/about.md ，打开 about.md 看下： 安装皮肤 到 皮肤列表 挑选一个心仪的皮肤，比如你觉得 Hyde 皮肤不错，找到相关的 GitHub 地址，创建目录 themes，在 themes 目录里把皮肤 git clone 下来： 创建 themes 目录 $ cd themes $ git clone https://github.com/spf13/hyde.git 运行Hugo $ ./hugo.exe server –theme=hyde –buildDrafts 部署 $ ./hugo.exe –theme=hyde –baseUrl=“https://605001659.github.io/circle/"
